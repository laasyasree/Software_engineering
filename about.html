About Us Page

An all-in-one online exchange web app for college students

This project is designed to be an all-in-one web application wherein, multiple options for buying, selling and renting things are provided. This website is especially relevant for hostel students and juniors, as they can use it to lend/sell their possessions to other students. This website will include the features like adding to cart, selling and renting with the additional option of renting your possessions for a period of time.
